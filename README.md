# System Performance Visualizer & CPU Scheduler Simulator

## Description
A desktop application developed using C++ and Qt to visualize system performance
and simulate CPU scheduling algorithms for educational purposes.

## Features
- CPU scheduling simulation (FCFS, SJF, Priority, Round Robin)
- Memory allocation visualization
- Real-time system performance monitoring (limited scope)
- Interactive GUI using Qt

## Technologies Used
- C++
- Qt Framework
- Linux (/proc filesystem)

## Academic Context
This project is developed as a Major Project for the Bachelor of Computer Applications (BCA)
program under Panjab University syllabus.
